Project 1- Just Breathe

by Yazhini Priyadharshini Rajesh

The idea of this project is to visualize annual air quality data, to see how air quality around the US has changed since the early 1980. The information is taken from the official site of the United States Environmental Protection Agency.

About the data The data was collected from 1980 to 2018 and are available online and the annual summary files are small enough to include all data in one file. About each file : Each group has data listed by year, in reverse order, back to 1980. Each table entry has the file name, linked to the file, the size of the (zipped) file, the number of data rows in the file, and the date the file was last modified. EPA updates these files twice per year; in the spring and fall (late May and November). Data collection agencies have up to 6 months to report their data.

The Data

This application visualizes the following:

  • About the Project: The user can choose any state from a list, and see all the counties in that state and then choose any of those counties, and be able to see an alphabetical list of all the counties in the US
  • AQI Air Quality: A pie chart showing the percentage of days, and a bar chart, and table showing the number of days where the AQI was good / moderate / unhealthy for sensitive / unhealthy / very unhealthy / hazardous. A line graph using the annual data from 1980-2018 showing lines for the median, 90th percentile, and max AQI over those years
  • Pollutants In the Air: A pie chart, bar graph and a table for each individual pollutant (CO, NO2, Ozone, SO2, PM2.5, PM10 ) showing the percentage of days in the year with that pollutant as the main pollutant. A line graph and table showing the percentages over the years for days CO / days NO2 / days Ozone, days SO2 / days PM2.5 / days PM10
  • Map View: of USA with the location of the county in picture.
  • We can also compare the data of 3 counties using the compare 3 counties option in the list.

Application

This application is written in R, using the shiny dashboard framework. Graphs use ggplot2 and plotly; the map view uses leaflet and location data from addTiles. In addition, tidyr and jpeg libraries are used to support corresponding functions.

It is designed to be viewed as a dashboard with no scrolling. Select the items on the left navigation pane to switch between various views. We can also minimize the sidepad by using the minimize button on top next to the title.

Items in the legend may be clicked to show/hide the item from the graph. Double-click on a legend item to show only that item. Hover over chart elements for more information about the point, bar or pie.

The theme of the project was obtained online through the dashboardthemes library.
